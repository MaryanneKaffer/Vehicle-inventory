# 🚘 Vehicle Inventory Management System

This is a **Full-Stack** vehicle inventory project based on a CRUD (Create, Read, Update, Delete) system. It allows you to register, view, and delete vehicles quickly and efficiently.

The main data request is implemented with pagination and includes advanced filters for name, brand, model, price, and year. To provide a better initial experience, the API comes with pre-registered vehicles, and the interface was designed to be clean, intuitive, and fully responsive.

This project was developed for educational purposes, focusing on mastering **Spring Boot** and integrating a robust back-end with a modern front-end ecosystem.

---

## 🛠️ Tech Stack

- **Back-end:** Java and Spring Boot.
- **Front-end:** React, TypeScript, Tailwind CSS, and HeroUI.
- **Image Management:** Cloudinary (API integration for uploads).
- **Deployment:** Vercel (Front-end) and Render (Back-end).

---

## ✨ Features

- [x] Full CRUD for vehicles.
- [x] Advanced filtering (brand, model, price, etc.).
- [x] Paginated data fetching.
- [x] Image upload via Cloudinary.
- [x] Responsive Design.
